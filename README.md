# sky-stones - 光遇国服红黑石日历

👉 **[在线使用](https://sky-stones.pages.dev)**

帮助《光·遇》国服玩家计算和查看**红石**与**黑石**降落时间的日历工具。

---

## 功能

- 日历视图，红石以红色标记，黑石以灰色标记
- 信息条实时显示当前时段状态及倒计时
- 红黑石进行中时，显示精确到秒的剩余时间
- 一键复制今日红黑石信息
- 日历导出为 PNG 图片
- 设备时间校准提示，避免因时钟偏差错过降落
- 支持 PC 端和移动端

---

## 技术栈

使用的外部库（通过 CDN 加载）：
- [Tailwind CSS](https://tailwindcss.com/) — 样式框架
- [Font Awesome](https://fontawesome.com/) — 图标
- [html2canvas](https://html2canvas.hertzen.com/) — 截图导出
